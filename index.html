@maplat/transform — デモ一覧

MaplatTransform の各機能を体験できるインタラクティブデモ

Phase 1 — 単一TIN

Single Map Demo

Transform クラスによる単一 TIN の双方向変換を可視化。
  • 無限遠半直線 / グリッド変換 / ラウンドトリップ誤差
  • v2 / v3 形式の切り替え
  • ズーム・パン対応の2座標系表示
→ demo/singlemap.html
Phase 2 — MapTransform / submap 選択

Submaps Demo

MapTransform クラスによる複数 TIN（sub_maps）の選択・変換を可視化。
  • priority / importance に基づくレイヤー選択
  • 各 submap の envelope・三角網を色分け表示
  • クリックで双方向の対応点を複数表示
  • v2 / v3 切り替え対応
→ demo/submaps.html
Phase 3・4 — ビューポート同期

Map Sync Demo

viewpoint2Mercs / mercs2Viewpoint を使って2つの絵地図間でビューポートを同期。
  • 館林御城図 ↔ 館林城下町図 を並べて表示
  • 片方のズーム・パン・回転がもう一方に伝搬
  • 中央固定の方角矢印（N）とスケール円
  • Envelope・三角網（TIN）表示
→ demo/mapsync.html
